What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation (layout, colors, fonts, etc.) of a document written in HTML. CSS helps you control the look and feel of your website, separate content from design, and make it more interactive and attractive.

Where CSS Fits in Web Development

CSS is applied to HTML elements to style them. It can be included directly in the HTML file (inline styles), embedded in the `` section of the document (internal styles), or referenced as an external file (external styles).

Commonly Used CSS Properties:

  • color - Sets the color of text.
  • background-color - Sets the background color of an element.
  • font-family - Defines the font for the text.
  • font-size - Specifies the size of the text.
  • margin - Defines the space around an element.
  • padding - Defines the space inside an element.
  • border - Adds a border around an element.

CSS Example:

/* This is an example of CSS */

/* Styling the body */
body {
    background-color: #f4f4f4;
    font-family: Arial, sans-serif;
}

/* Styling the header */
h1 {
    color: #2c3e50;
    text-align: center;
}

/* Styling the paragraphs */
p {
    font-size: 16px;
    color: #333;
}

/* Styling the link */
a {
    color: #2980b9;
    text-decoration: none;
}
